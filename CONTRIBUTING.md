# 贡献指南

欢迎为 Markdown Theme Hub 贡献主题资源！

## 如何贡献

### 添加新主题

1. Fork 本仓库
2. 编辑 `README.md`，在对应分类下添加主题
3. 按以下格式添加：

```markdown
| [主题名称](GitHub链接) | 编辑器 | 简短描述 |
```

示例：
```markdown
| [typora-vue-theme](https://github.com/blinkfox/typora-vue-theme) | Typora | Vue 文档风格 |
```

### 添加新分类

如果你的主题不属于现有分类，可以创建新分类：

1. 在 `README.md` 中添加新分类标题
2. 在 `CATEGORIES.md` 中同步添加
3. 提交 PR

### 提交 PR

1. 确保格式与现有内容一致
2. PR 标题格式：`Add: 主题名称` 或 `Update: 分类名称`
3. 简要说明主题特点

## 收录标准

- ✅ 开源或有公开下载链接
- ✅ Markdown 编辑器相关主题
- ✅ 有明确的作者/来源
- ❌ 收费且无免费试用
- ❌ 无明确来源
- ❌ 已停止维护且无法使用

## 联系方式

如有问题，请提交 Issue。
