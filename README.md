## 🛠️ 一键安装脚本

### Typora 主题安装

```bash
# 安装 Vue 主题
curl -L https://github.com/blinkfox/typora-vue-theme/archive/refs/heads/master.zip -o vue-theme.zip
unzip vue-theme.zip -d ~/Library/Application\ Support/abnerworks.Typora/themes/ 2>/dev/null || \
unzip vue-theme.zip -d ~/.config/Typora/themes/ 2>/dev/null
rm vue-theme.zip
```

### Obsidian 主题安装

Obsidian 主题可直接在应用内安装：
1. 设置 → 外观 → 主题 → 管理
2. 搜索主题名称
3. 点击安装并启用

## 📥 如何安装主题

### Typora 主题安装

1. 下载主题 CSS 文件
2. 打开 Typora → 偏好设置 → 外观 → 打开主题文件夹
3. 将 CSS 文件复制到主题文件夹
4. 重启 Typora，从菜单选择主题

### Obsidian 主题安装

**方法一：应用内安装（推荐）**
1. 设置 → 外观 → 主题 → 管理
2. 搜索主题名称
3. 点击安装并启用

**方法二：手动安装**
1. 下载主题文件
2. 打开设置 → 外观 → 打开主题文件夹
3. 将主题文件夹复制到 `themes` 目录
4. 重启 Obsidian

### MWeb 主题安装

1. 下载主题 CSS 文件
2. 打开 MWeb → 偏好设置 → 主题 → 打开主题文件夹
3. 将 CSS 文件复制到主题文件夹
4. 从主题菜单选择

### Logseq 主题安装

1. 打开 Logseq → 设置 → 通用 → 主题
2. 点击 "Open folder containing custom.css"
3. 将主题 CSS 内容复制到 `custom.css`
4. 或安装 [Logseq Marketplace](https://github.com/pengx17/logseq-market) 插件
