## 📥 如何安装主题

### Typora 主题安装

1. 下载主题 CSS 文件
2. 打开 Typora → 偏好设置 → 外观 → 打开主题文件夹
3. 将 CSS 文件复制到主题文件夹
4. 重启 Typora，从菜单选择主题

### MWeb 主题安装

1. 下载主题 CSS 文件
2. 打开 MWeb → 偏好设置 → 主题 → 打开主题文件夹
3. 将 CSS 文件复制到主题文件夹
4. 从主题菜单选择
