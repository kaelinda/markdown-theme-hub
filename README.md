## 🛠️ 一键安装脚本

### Typora 主题安装（macOS）

```bash
# Vue 主题
cd ~/Library/Application\ Support/abnerworks.Typora/themes/
curl -LO https://raw.githubusercontent.com/blinkfox/typora-vue-theme/master/vue.css
```

### Obsidian 主题安装

应用内直接安装：设置 → 外观 → 主题 → 管理 → 搜索安装
