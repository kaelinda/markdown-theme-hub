```css

npm install --save github-syntax-light
npm install --save github-syntax-dark

@import "github-syntax-dark/lib/github-dark.css";
```

## 🔧 使用建议

### 推荐组合

| 场景 | 推荐主题 |
|------|----------|
| 日间工作 | GitHub Light Default |
| 夜间编程 | GitHub Dark Default |
| 护眼模式 | GitHub Dark Dimmed |
| 可访问性需求 | High Contrast / Colorblind 变体 |
| 跨平台一致 | VS Code + Terminal + Obsidian 全套 GitHub 主题 |

### 与其他配色对比

| 配色 | GitHub 对比 |
|------|-------------|
| **Catppuccin** | GitHub 更专业，Catppuccin 更柔和 |
| **Dracula** | GitHub 更低调，Dracula 更个性化 |
| **Nord** | GitHub 更中性，Nord 更冷淡 |
| **Tokyo Night** | GitHub 更经典，Tokyo Night 更现代 |

---

> 💡 **提示**: GitHub 主题是开发者最熟悉的界面风格，适合追求专业、熟悉感的用户。官方 VS Code 主题支持多种变体，满足不同可访问性需求。
