## 🎨 配色选择指南

### 按使用场景

| 场景 | 推荐配色 |
|------|----------|
| 长时间编程 | Nord, Catppuccin Mocha |
| 夜间工作 | Dracula, Tokyo Night |
| 日间写作 | Catppuccin Latte, GitHub Light |
| 学术论文 | GitHub Light, Solarized Light |
| 知识管理 | Rosé Pine, Catppuccin |
| 终端开发 | Dracula, Nord |

### 按个人偏好

| 偏好 | 推荐配色 |
|------|----------|
| 柔和护眼 | Catppuccin, Rosé Pine |
| 高对比度 | GitHub Dark, Material Darker |
| 冷色调 | Nord, Tokyo Night |
| 暖色调 | Gruvbox, Dracula |
| 专业简约 | GitHub, One |
| 独特个性 | Tokyo Night, Gruvbox |
