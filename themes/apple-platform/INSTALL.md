# Apple 风格主题安装方案

## 推荐安装方式

### 方案：mdnice / 自定义 Markdown 编辑器粘贴 CSS

**推荐文件：** `apple-notes-light.css`

### 为什么推荐这个方案
- 零安装成本
- 可实时预览
- 可以快速切换为 Apple 风格
- 更适合公众号和内容平台

## 操作步骤

1. 打开 Markdown 编辑器（如 mdnice）
2. 进入「自定义主题」
3. 复制 `apple-notes-light.css` 全部内容
4. 粘贴并保存
5. 预览后复制到目标平台

## 文件选择建议

- `apple-notes-light.css`：默认推荐，长文阅读
- `cupertino-clean.css`：更高级、更像发布页
- `apple-notes-dark.css`：暗色阅读

## 不推荐的安装方式

### 1. CDN 外链
**不推荐原因：** 内容平台通常不支持外链 CSS。

### 2. npm 包方式
**不推荐原因：** 对内容创作用户太重，安装链路不优雅。

## 一句话结论

如果你追求“丝滑优雅”，就用：

**mdnice + apple-notes-light.css**
